# SymfonyMessenger Module
[![Latest Stable Version](https://poser.pugx.org/spryker/symfony-messenger/v/stable.svg)](https://packagist.org/packages/spryker/symfony-messenger)
[![Minimum PHP Version](https://img.shields.io/badge/php-%3E%3D%208.3-8892BF.svg)](https://php.net/)

SymfonyMessenger module provides integration with Symfony Messenger for Spryker applications.

## Installation

```
composer require spryker/symfony-messenger
```

## Documentation

[Spryker Documentation](https://docs.spryker.com)
Refer to [Symfony Messenger documentation](https://symfony.com/doc/current/messenger.html) and Spryker documentation for usage details.
